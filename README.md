# verivox-tests

Tests for website ```https://www.verivox.de``` based on WebdriwerIO and Cucumber.

## Requirements

- node 10.16.0+
- npm 6.9.0+
- java 8+

## Setup

Navigate to project directory.

Install dependencies

```bash
$ npm install
```
## Tests

Run tests

```bash
$ npm test
```

**Note**: Allure reports regenerate automatically and opens in your browser.
